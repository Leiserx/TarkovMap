# TarkovMap
塔科夫地图工具，原来自用的适配截止于26年1月。
联机功能的话只写了linux服务器的一键脚本，需要windows用的话可以搭配内网穿透和让ai手搓一个windows的websocket服务就行了。具体的接口传参格式参考下服务端的sh脚本。

逃离塔科夫辅助工具使用手册
本工具旨在为玩家提供高效的地图交互、任务追踪及多人联机位置共享功能。

⚙️ 设置页面介绍
用户名称：设置后，在多人联机模式下的地图上会展示你的实时位置与名称。

拓展定位网页：建议在局域网内使用。通过平板或手机浏览器打开，即可同步查看玩家位置及任务目标信息。

派系选择：用于区分派系专属任务，根据实际情况选择即可。

多人模式：

前提条件：需先进行服务端部署。

功能说明：连接服务器后可查看队友位置。

截图目录：

原理：监听塔科夫截图目录并解析位置。

注意：程序会自动清理过往截图（仅保留最近几十张）。开启多人模式后，截图会自动上传至服务器以实现位置共享。

游戏日志目录：

功能：监听日志以实现自动切图（根据当前地图自动切换交互地图）。

状态：支持在线和离线模式。注：如遇匹配机制变更，功能稳定性可能受影响。

窗口控制：支持一键弹出和一键隐藏（快捷键操作）。

缓存管理：管理任务完成记录等缓存数据。请谨慎清理，数据清除后不可恢复！

瓦片数据：本地缓存卫星地图数据，减少对 tarkov.dev 的重复请求，提升加载速度。

🛠️ 工具箱
1. 屏幕亮度
功能：与“小蝴蝶”类似，支持多显示器调节。

注意：在全屏模式下不生效。若在无边框窗口模式下使用，必须以管理员身份启动本软件。

2. 跳蚤市场
数据：数据每 30 分钟左右更新一次，具有较强的参考价值。

模式切换：通过右上角按钮可在 PVE 和 PVP 市场数据间自由切换。

3. 藏身处
提供基础的藏身处建设及需求参考。

📑 任务系统
本工具任务数据同步自 tarkov.dev。如需更详细的胎教级攻略或关系图，建议参考“大傻哥”的相关资料。

核心功能：
追踪功能：点击任务中的“追踪”，地图上会即时标记任务坐标或任务范围。

快捷入口：追踪任务后，交互地图右侧会出现书本标记，左键点击可查看当前所有追踪任务。

目标隐藏（踩点标记）：

点击地图右侧的眼睛图标进入隐藏模式。

在此模式下点击地图上的任务目标，该图标会隐藏（适用于记录多点任务的进度）。

找回隐藏点：若点错，右键点击书本标记打开隐藏列表菜单，点击对应的小眼睛即可重新显示。

🗺️ 交互地图
撤离点查看：通过右下角悬浮球可快速切换/展示撤离点。

地图模式：支持切换 SVG 矢量图、卫星图或平面图纸。

快捷操作：

颜文字图标：点击可手动重连多人服务端。

自动切图：颜文字图标同时代表自动切图开关（需正确设置日志监听）。

图层切换：按住 CTRL + 鼠标滚轮 可快速切换地图图层。

温馨提示：部分任务数据（如试驾任务）可能存在延迟，具体请以游戏内实际数据为准。
