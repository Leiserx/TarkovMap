## 设置页面介绍
用户名称：这个主要是用来多人联机时地图上会展示出你的位置+名称。

拓展定位网页：我倾向于局域网内使用，用平板浏览器打开可以看到玩家位置和任务位置信息。
<img width="2560" height="1439" alt="image" src="https://github.com/user-attachments/assets/020b667c-53db-452f-ac28-fe1ec791fffd" />

派系选择：有些任务是派系专属的，随便选都行。

多人模式：要看别人的位置需要先服务端部署，然后连服务才行。

截图目录：这个就是塔科夫截图目录，监听这个目录来解析位置的（有自动清理，会保留最近几十张图忘记了。开了多人模式会上传的服务器分发给别人来共享位置）。

游戏日志目录：设置了在互动地图页面打开自动切图，会监听日志实现换图自动切换，在线和离线都可以。（不确定可以用不，因为上次能用还是1月份，最近好像匹配机制变了但是日志不知道有没有变化）

窗口控制：这个就是一键弹出和一键隐藏（我觉得不好用）。

缓存管理：就是任务完成记录的缓存，清了就没了。清理前请深思！！！

瓦片数据：用来缓存卫星图的，防止重复向 tarkov.dev 请求地图就缓存了。

## 工具介绍
屏幕亮度：自己设置一下和小蝴蝶一样用，支持多显示器，但是全屏下不生效，无边框下要在游戏内生效必须开管理员模式启动软件。

其他功能：原本还有健身和识别物价的但是感觉不要用删除了，可以用大傻哥的自动健身脚本。

## 跳蚤市场
就是跳蚤市场，数据一般时效性是半小时左右，可以用做参考。

右上角有个 PVE 和 PVP 切换按钮，显示 PVE 就是 PVE 的跳蚤，显示 PVP 就是 PVP 的。

## 任务
就正常的任务功能。但是没做关系图，挺麻烦的可以看大傻哥的。

要详细攻略也看大傻哥的，胎教级教程，我这只有坐标，而且任务用的 tarkov.dev 的数据，试驾任务我看也没更新，具体以游戏为准。

追踪功能：

任务里面点下追踪，对应地图就展示出任务的标记了。如果任务有完成范围的话点一下就会展示出任务范围。

当你有追踪任务时，互动地图右侧会多一个书本标记，左键点击可以看到你当前追踪的任务。

目标隐藏功能：

互动地图右侧会有一个眼睛，点一下会发光，这个时候点下地图上的任务目标，就会隐藏以免忘记哪些点踩了哪些没踩。

如果你发现点错了，那就右键下书本，会有个隐藏列表菜单，在点下里面的小眼睛就可以重新看到了。

## 互动地图
右下角悬浮球：可以切换展示撤离点的位置，还能切换地图类型（SVG 或 卫星图），还有个平面图纸可以看看。

重连与切图：那个颜文字点了会自动重连多人服务，开不开启自动切图就看那个按钮（前提必须设置日志监听）。

快捷键：按住 CTRL + 滚轮 可以切换地图图层。
